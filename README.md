# File Automation Project

## Description
This project is a Python automation script that performs file operations such as creating, reading, renaming, moving, and cleaning files. It also includes logging and user input support.

## Features
- File creation and reading
- File renaming and moving
- CSV file processing
- Logging system (log.txt)
- User input support
- Optional file cleanup

## How to Run
1. Run the Python script
2. Enter folder name when prompted
3. Choose whether to clean .txt files (yes/no)

## Sample Input
file_automation_demo  
yes

## Sample Output
- Creates folder  
- Reads and processes files  
- Moves/renames files  
- Displays total tasks  
- Shows final files
